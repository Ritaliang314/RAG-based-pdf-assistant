# RAG Based PDF Assistant


- Streamlit
- LangChain
- Google Gemini
- FAISS

## Features

- Upload PDF
- Ask questions
- Get answers based on document content

## Setup

please clone this repository and install the requirements:
```
pip install -r requirements.txt
```
Add your Google API key to the `.env` file.


## Usage


```
streamlit run app.py
```
